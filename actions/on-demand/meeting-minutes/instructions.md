# Meeting Minutes

An example on-demand action showing the standard bench action structure.

## How to use

1. Drop source material (audio transcript, agenda PDF, screenshots) into `assets/`.
2. Ask Claude to follow these instructions.

## Task

Read all files in `assets/` and produce structured meeting minutes including:
- Date and participants
- Key decisions made
- Action items (owner + deadline)
- Open questions

Format the output as a dated markdown file: `assets/YYMMDD-minutes.md`.
